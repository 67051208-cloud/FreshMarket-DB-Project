# 🛒 FreshMarket (POS & Delivery System)
# สามารถเข้าใช้งานได้ตามลิงค์นี้ครับ : https://freshmarket-db.great-site.net/

ระบบจัดการร้านค้าสด มีทั้งระบบหน้าร้าน (POS) และระบบสั่งซื้อออนไลน์ (Delivery) 
พัฒนาด้วย HTML, CSS, JavaScript, PHP และเชื่อมต่อกับฐานข้อมูล MySQL แบบ Full System

## ✨ ฟีเจอร์เด่นในระบบ
* **ระบบ POS หน้าร้าน:** สำหรับพนักงานแคชเชียร์ คิดเงินรวดเร็ว
* **ระบบ Delivery:** สำหรับลูกค้า สั่งออนไลน์พร้อมคำนวณค่าจัดส่ง 30 บาทอัตโนมัติ
* **ระบบชำระเงิน:** รองรับการจ่ายเงินสด, เก็บเงินปลายทาง (COD) และ **สแกนจ่ายผ่าน QR Code** (มีระบบ Gen QR Code อัตโนมัติในบิล)
* **ระบบ Inventory:** ตัดสต๊อกสินค้าและคำนวณต้นทุน/กำไร
* **ระบบ Review:** ลูกค้าสามารถให้คะแนนดาวและเขียนรีวิวสินค้าได้

---

## 🛠️ วิธีการติดตั้งและรันโปรเจกต์ 
1. นำไฟล์ทั้งหมดใน Repository นี้ไปวางไว้ในโฟลเดอร์ `htdocs` ของโปรแกรม XAMPP (แนะนำให้ตั้งชื่อโฟลเดอร์ว่า `freshmarket`)
2. เปิดโปรแกรม XAMPP Control Panel แล้วกด Start ที่โมดูล **Apache** และ **MySQL**
3. เปิดเบราว์เซอร์แล้วเข้าไปที่ `http://localhost/phpmyadmin/`
4. สร้างฐานข้อมูลใหม่ (Database) ชื่อว่า **`freshmarket_db`** *(สำคัญ: ต้องตั้งชื่อนี้เท่านั้นเพื่อให้โค้ดเชื่อมต่อได้)*
5. เข้าไปที่ฐานข้อมูล `freshmarket_db` เลือกแท็บ **Import** (นำเข้า) แล้วอัปโหลดไฟล์ `freshmarket_db.sql` ที่แนบมาในโปรเจกต์นี้ เพื่อสร้างโครงสร้างตารางและข้อมูลจำลองทั้งหมด
6. เปิดเบราว์เซอร์แล้วเข้าไปที่ `http://localhost/freshmarket/index.html` เพื่อเริ่มใช้งานระบบได้ทันที!

## 👥 ข้อมูล Users สำหรับทดสอบล็อกอิน
* **เข้าระบบพนักงาน (POS):** ใช้ ID `EMP-001`
* **เข้าระบบลูกค้า (Delivery):** ใช้ ID `USR-101`
* หรือสามารถทดลอง **"สมัครสมาชิกใหม่"** ผ่านหน้าเว็บได้เลย ข้อมูลจะถูกบันทึกลง Database จริง
